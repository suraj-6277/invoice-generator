# Invoice Generator

A simple invoice generator for freelancers and small businesses to quickly create professional invoices.

---

## 🚀 Preview

![Preview](./public/og-image.jpeg)

---

## ✨ Features

- Create professional invoices easily  
- Fast and simple UI  
- Download invoices as PDF  
- Fully customizable  
- Unlimited invoice generation  

---

## 🛠️ Tech Stack

- Next.js  
- React PDF  
- Tailwind CSS  

---

## ⚙️ Run Locally

```bash
git clone https://github.com/suraj-6277/invoice-generator.git
cd invoice-generator
npm install
npm run dev
